## 🔥 Iniciando o projeto

Primeiro, instale as dependências necessarias com o comando: 

```bash
npm install
```
  
Em seguida, execute o servidor de desenvolvimento:

```bash
npm run dev
# ou
yarn dev
# ou
pnpm dev
# ou
bun dev
```

Abra [http://localhost:3000](http://localhost:3000) com seu navegador para ver o resultado.
